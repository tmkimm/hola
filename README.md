## 🔖 소개

**Hola**는 개발자들을 위한 플랫폼으로, 모든 개발자가 스터디 및 사이드 프로젝트를 진행할 동료를 쉽게 구할 수 있도록 플랫폼을 제공합니다.🎉

## 📚 기술스택

| 분야           | 사용 기술                       | 비고 |
| -------------- | ------------------------------- | ---- |
| FrontEnd       | React, javascript               |
| BackEnd        | Express                         |
| Database       | MongoDB(Mongoose)               |
| Cloud Services | AWS EC2, AWS Cloudfront, AWS S3 |
| TOOL           | VSCODE, Notion, Slack           |
| DESIGN         | Figma                           |
